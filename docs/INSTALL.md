# Installation

### Requirements
All the codes are tested in the following environment:
* Linux (tested on Ubuntu 22.04)
* Python 3.9.19
* PyTorch 1.8.1
* CUDA 11.1
* [`spconv-cu111`](https://github.com/traveller59/spconv)
* gcc version 9.4.0
* Encourage you to try other settings!

### Install the dependent libraries as follows:

* Install the python dependent libraries.
  ```shell
    pip install -r requirements.txt 
  ```

### Install GCFS-3Det
Please install re-install `GCFS-3Det` by running `python setup.py develop`.
